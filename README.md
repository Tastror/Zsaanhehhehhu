# 上海闲话 Zsaanghehhehhu

## 运行

运行 GUI

```python
python zsaanghehhehhu.py
```

查看所有可能音（包括该方案下的偶然缺失）

```python
python zsaanghehhehhu.py --testhanzi
python zsaanghehhehhu.py --testphonology
```

## 查看音系结构

```python
python analyze/plot_grid.py
python analyze/analyze.py
python analyze/group_analyze.py
```

## 功能

一个 GUI，输入文字（序列），输出对应的

1. IPA
2. T拼
3. 吴语学堂拼音
4. 吴语协会拼音

多个文白读/多音字/异体字的，显示所有情况。如果没有结果显示暂空。

搜到以后存到本地 json，优先直接从本地读，不仅方便后续查询，也方便暂空的手动填补。

> `readings.json` 已将 2026.4.20 前吴语学堂 (<https://www.wugniu.com>) 的所有繁体字与传承字保存完毕，且有部分简体字与异体字或新字添加。

## 音标对照表

### Tastror 拼音方案（简称：T拼）

规范：以拼音为基础，h表浊音，q表喉塞，ê表/ɪ//e//ᴇ//ɛ/附近读音，e表/ə//ɤ/附近读音，ü不省写两点。

符号 ' 一般写作 '，不方便使用的场合可用 hh；ê 一般写作 ê，不方便使用的场合可用 e；ü 一般写作 ü，不方便使用的场合可用 yu。本文一律用 '、ê 和 ü。

Ø 表示什么都不填，后同

- 声母

  帮b、滂p、平bh、明m、敷f、奉v  
  端d、透t、定dh、南n(娘n)、来l  
  照z、穿c、审s、禅zs  
  见j、溪q、群jh、心x、谢xh  
  光g、框k、狂gh、牙ng  
  影Ø、好h、匣' \[转写hh\]

- 介音

  Ø、i、u

- 韵母

  鞋a、海ê \[转写e\]、看oe、少o、口eu、瓜u、布uu、低i、虑ü \[转写yu\]、试y  
  打ang、当aang、春eng、中ong、英ing、君üing \[转写yuing\]  
  法aq、出eq、角oq、力iêq \[转写ieq\]、决üêq \[转写yueq\]  
  而er、亩m、唔n、五ng

- 声调

  阴平（1）（调值 52）：à  
  阴上去（通归为阴去 5）（调值 34）：ā  
  阳平上去（通归为阳去 6）（调值 23）：á  
  阴入（7）（调值 5）：aq  
  阳入（8）（调值 12）：áq

注1：音位上的 i + oe 实际形成 ü + oe /yø/，这里为减少不必要介音，在后续音位分析时以 ioe 计，但最终读音和 GUI 展示均为 üoe。

注2：一般情况下，声调标在韵母（而非介音）的第一个字符上，除了 iêq、üêq 标在 ê 上。特殊字符仍然正常标注，例如 sȳ、lǜ、hê̄ 等。ü 上有声调且转写 yu 时标在 u 上。

注3：新派上海话

- 可能不区分 ang(冷)/aang(浪)、uoe(官)/oe(干)、üoe(娟)/ü(居)；
- 后高元音 u(抓)/uu(做) 通过摩擦区分，后者有摩擦（也可不区分）；
- 入声 aq(客)/eq(克) 的开口较为近似，前者比后者略开（也可不区分）；
- 浊咝音 xh(谢/前)、zs(禅/上) 是高气流的浊音甚至清音（不过气流仍然比清音要少），这也是此处使用 zs 表示 /z/ 的另一层原因；
- '(匣)的浊流几乎消失，使其和Ø(影，零声母)除了声调以外几乎没有区别。

## 音系结构

声母/韵母使用**T拼**，声调使用八调标记（1~8 对应 阴平/阳平/阴上/阳上/阴去/阳去/阴入/阳入，上海话实际只使用了其中五个，1阴平/5阴去/6阳去/7阴入/8阳入）。数据来源为吴语学堂。

### (1) 声母 × 声调

上海话的「阴阳调分化」非常整齐。把声母按历史清浊分四类即可覆盖全部格局。

| 类别 | 声母（T拼） | 非入声声调 | 入声声调 |
| ------ | ------------ | ------------ | ---------- |
| 清声母 | `b p f d t z c s j q x g k h` | 1/5 | 7 |
| 全浊 | `bh v dh zs jh xh gh '` | 6 | 8 |
| 次浊 `m` | `m` | 1/5/6 | 7/8 |
| 次浊鼻边 | `n(南) n(娘) l ng` | 1/6（`n`(娘) 仅有 5 调特例 `niāng`「仰」） | 8 |
| 零声母 | `Ø` | 舒声 1/5；成音节 `m/n/ng` 1/6 | 7 |

T拼中的 `n` 同时是南母和娘母，不接 i 介音为南母 /n/，接 i 介音为娘母 /ɲ/。后文将两者分开。

具体细分表如下

× = 该声调有字；· = 该声调无字；△ = 局部类别/词汇特例

**系统结论**：

- **清声母**（全清 + 次清，14 个）只可为阴调 {1, 5, 7}。
- **全浊**（8 个）只可为阳调 {6, 8}。
- **次浊`m`** 可为所有声调，即 {1, 5, 6, 7, 8}。
- **其他次浊`n`(南)/`n`(娘)/`l`/`ng`** 只可为阴平/阳去/阳入，即 {1, 6, 8}。存在一个5调特例「仰」`niāng`。
- **零声母** 不考虑单韵母 `m/n/ng` 时只可为阴调。具体而言，普通舒声只可为 {1, 5}；入声只可为 7。特例为单韵母 `m/n/ng`，只可为 {1, 6}。他们可以视为单独的次浊声母+零韵母，此时零声母可以直接合并进清声母。

### (2) 介音 × 韵母

共 43 对（共 4×25 = 100 个组合中的）：

| 介音 | 可接韵母（T拼） | 合计 |
| ------ | -------------- | ------ |
| `Ø` | `a, ê, oe, o, eu, u, uu, i, ü, y, ang, aang, eng, ong, ing, üing, aq, eq, oq, iêq, üêq, er, m, n, ng` | 25 |
| `i` | `a, ê, oe, o, eu, ang, aang, ong, aq, oq` | 10 |
| `u` | `a, ê, oe, ang, aang, eng, aq, eq` | 8 |

**系统结论**：

- 空介音 `Ø` 与全部 25 个韵母相容。
- `i` 介音只与 **{`a, ê, oe, o, eu, ang, aang, ong, aq, oq`}** 这 10 个韵母相配。
- `u` 介音只与 **{`a, ê, oe, ang, aang, eng, aq, eq`}** 这 8 个韵母相配。

### (3) 声母组 × 介音 × 韵母

按「声母自然类」归纳。28 个声母（含零声母 Ø）按发音部位 / 方式划为 12
组；**同组内部规则应完全一致**——组内至少一个声母在某 `(介音, 韵母)`
格子里有字例，就视为整组规则合法；组内全无字，才算组级系统缺失。
个别成员落入"允许"格却无字例，计作偶然缺失，不纳入 `_is_sensible_combo`
硬规则。

#### A. 声母自然类（12 组）

| 组 | 声母（T拼） | 描述 |
| ---- | ---- | ---- |
| G1 | `b p bh` | 唇塞 |
| G2 | `m` | 唇鼻 |
| G3 | `f v` | 唇齿擦 |
| G4 | `d t dh` | 齿塞 |
| G5 | `n`(南) `n`(娘) | 齿鼻、齿腭鼻 |
| G6 | `l` | 边音 |
| G7 | `z c s zs` | 齿擦、塞擦 |
| G8 | `j q jh x xh` | 腭塞擦、腭擦 |
| G9 | `g k gh h` | 软腭塞、软腭擦 |
| G10 | `ng` | 软腭鼻 |
| G11 | `'` | 浊喉擦 |
| G12 | `Ø` | 零声母 |

> T拼 把南母和娘母都写成 `n`：跟 `i` 介音拼的那层读 /ɲ/（娘），不跟 `i` 拼
> 读 /n/（南）。组别归类按发音部位——南母和娘母都归 G5（齿鼻），
> 只在 i 介音下实现为 /ɲ/。疑母 `ng` 虽也是软腭部位，但作为唯一的
> 软腭鼻音，分布跟阻塞的 G9 不同（详见下文 B/D 表），独立成 G10。
> 边音 `l` 虽跟南/娘共享齿部位，但它本身不带鼻音／腭化，分布也跟
> `n(南)/n(娘)` 明显不一致（见 B/C 表），独立成 G6。

> **G5 组内 `n(南)/n(娘)` 互补分布（吴拼层面）**：
> G5 外围的"组级"规则仍作整体看，但组内 `n`(南) 和 `n`(娘) 是完全互补的，
>
> - `n`(娘) /ɲ/ 只出现在 **i 起头** 的音节，即 `med = 'i'` **或** `fin ∈ {i, ü, ing, üing, iêq, üêq}`。
> - `n`(南) /n/ 只出现在 **非 i 起头** 的音节。唯一例外是 `n(南)i` /ni/ 的特殊文读，只有唯一汉字「你」，T拼里将其与 `n(娘)i` /ɲi/ 合流不再区分。

**横向系统结论：**

- **舌尖元音 `y` /ɿ/ 只见于 G7**（`z/c/s/zs`）；其余 11 组全无。
- **`er` 只见于 G11**（`'`）；其余 11 组全无。
- **自成音节 `m / n / ng` 只见于 G12**（零声母）；其余 11 组全无。
- **撮口 `ü / üing / üêq`** 只见于 G5（ü/üêq）+ G6（仅 ü）+ G8 + G11 + G12；
  G1 / G2 / G3 / G4 / G7 / G9 / G10 全无 —— 唇音 / 齿塞 / 齿擦 / 见组 / 疑母
  都跟 /y/ 不相容。
- **腭音组 G8 空介音下只接 `i / ü` 系韵母** `{i, ü, ing, üing, iêq, üêq}`；
  其余 19 韵全无（腭音本身含 /j/，必须后接前高元音落脚）。
- **`i / ing / iêq` 系韵母** 在 G7（齿擦）/ G9（见组）/ G10（疑）下整体全无：
  这些组 + `/i/` 会腭化改写为 G8 声母（见组 / 晓）或 G5 娘母（疑）；G3
  额外在 `ing / iêq` 也系统缺失。
- **G5 `n(南)/n(娘)` 与 G6 `l` 对照**：空介音下 G5 独占 `{u, üêq}`
  （`nú`挪 / `nüếq`月），G6 独占 `{eu, ang}`
  （`léu`楼 / `láng`冷）。
- **G10 `ng` 跟 G9 `g/k/gh/h` 对照**：阻塞的 G9 允许 `eng / ong`
  （`gèng`跟 / `gòng`公），G10 `ng` 这两韵全无。

**系统结论：**

- **G7（齿擦 / 塞擦）、G9（见 / 晓）、G10（疑）整组都不接 i 介音** ——
  G7 / G9 被腭化为 G8；G10 `ng + i` 则变为 G5 的 `n`(娘)。
- **唇音阻塞组（G1 `b/p/bh`、G3 `f/v`）的 i 介音只保留 `o` 一韵**
  （`biō`表 / `vió`覅）。
- G2 `m` 额外多 `eu`（`miéu`谬），G4 `d/t/dh` 再多 `a`（`dià`爹）。
- **G5 `n(南)/n(娘)` 与 G6 `l` 在 i 介音下同样分化**：G5 覆盖 8 韵，G6 只
  4 韵（共有 `{o, eu, ang, aq}`，G5 额外独占 `{ê, oe, ong, oq}`）——
  娘母比来母在 i 介音下分布更广。
- G8 / G11 / G12 （非阻塞组）的 i 介音分布比较接近 (2) 中列出的完整
  i-med 韵母集 `{a, ê, oe, o, eu, ang, aang, ong, aq, oq}`。

#### D. `u` 介音

| 组 + `u` | 可接韵母（T拼） |
| ---- | -------- |
| G1-G8 | **系统级全无** |
| G9 `g/k/gh/h` | `a, ê, oe, ang, aang, eng, aq, eq`（8） |
| G10 `ng` | **系统级全无** |
| G11 `'` | `a, ê, oe, ang, aang, eng, aq, eq`（8） |
| G12 `Ø` | `a, ê, oe, ang, aang, eng, aq, eq`（8） |

**系统结论：**

- **u 介音只出现在软腭塞 + 晓（G9）/ 匣（G11）/ 零声母（G12）后**。其他
  24 个声母（含疑母 `ng`、边音 `l`）都不配合口。
- G9 / G11 / G12 三组的 u 介音韵母分布**完全一致** —— 整齐允许
  `{a, ê, oe, ang, aang, eng, aq, eq}`，其他韵一概不出现。
- G10 疑母 `ng` 虽和 G9 同为软腭，但 **`ng` 不可带 u 介音**。

#### E. 该方案下的偶然缺失

以上三张表按"组级"划线，若组内某个声母落入"允许"的 `(组, 介音, 韵母)` 里却无字例，则记为该方案下的偶然缺失。最终结果为
**810 个合法 (声母, 介音, 韵母, 声调) 表层音节**，其中
**693 个（约 86%）** 在吴语学堂数据库中能找到字，其余 ~14% 属该方案下的偶然缺失。
